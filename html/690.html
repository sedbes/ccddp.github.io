《Brain Rules》（大脑规则）一书的12条原则

时间：2023-02-08 20:20:31  来源：战隼  作者：

关键词：大脑开发 | 学习方法

这几天每天晚上都能去书店看一会书，到现在为止一共看了６、７本。包括《影响力2》、《谁杀了我的牛》、《全世界最简单解决问题的办法》、《对答如流》、《粮食战争》、《求医不如求已3》、《你的记忆怎么了》。更高兴的是这几本书质量相对高一些，从中学到了不少新的知识。每当学到有用的知识时，我的心情就很好。过几天，我会整理一下然后把与学习有关的阅读心得放上来。

前段时间跟朋友聊天，朋友给推荐了一本他现在正在阅读的书《Brain Rules》（大脑规则），并聊了一下阅读心得．听完之后很感兴趣，到网上搜索了下，虽然没有找到电子书下载，但找到了作者单独为这本书建立的网站，网站有大量关于这本书籍的内容介绍和相关视频．有时间可以看一下，在youtube上还作者John Medina上传的与这本书有关的视频，很有趣．点击这里：Brain Rules

约翰．梅狄纳（John Medina）博士是一位专注于人脑发展基因以及精神病遗传学问题的发展分子生物学家。兼任华盛顿大学医学院和西雅图太平洋大学双聘教员。他在书．

2008年3月18日，梅狄纳出版新作《大脑规则》（Brain Rules），Pear Press专门为这本新书开发了网站(www.brainrules.net)：大脑规则：生存与发展与工作、家庭和学校的12条原则（Brain Rules：12 Principles for Surviving and Thriving at Work, Home and School）。在这个网站上，包含了大量有关这本著作介绍的视频、音频。

他在这本新书中提出的12条大脑原则有：

大脑原则1：运动（EXERCISE | Rule #1: Exercise boosts brain power.）

运动对大脑效能有相当显著的提升效果，想要变得更聪明，就要在会议或工作等行程之中，排入运动的时间，你的思绪将会变得更清晰。

大脑原则2：进化（SURVIVAL | Rule #2: The human brain evolved, too.）

人类最聪明的物种，但并不是最强壮的。我们能够大量繁衍，是因为能从错误中学习、解决问题并且团结合作。要成功就必须与他人建立良好关系。

大脑原则3：大脑回路（WIRING | Rule #3: Every brain is wired differently.）

地球上每个人的大脑回路或多或少都有些不同，这表示每个人都有自己的学习方式。如果不保持这种弹性去接受差异，就无法充分运用认知能力。

大脑原则4：注意力集中的时间（ATTENTION | Rule #4: We don’t pay attention to boring things.）

我们不会注意无聊的事物，而且在同一时间也只能专注于一项事物。别想要多任务作业，那是不可能的，要专注于目前正在进行的事物。

大脑原则5：短期记忆（SHORT-TERM MEMORY | Rule #5: Repeat to remember.）

人脑一次只能记住7项事物，而且只能维持30秒左右，因此要学习东西，就要一再重复。经过不断的重复，将要学习的事物嵌入脑海中。

大脑原则6：长期记忆（LONG-TERM MEMORY | Rule #6: Remember to repeat.）

如果想记住某件事，就要定期重复接触这件事。人脑需要多年的时间才能牢牢记住一件事，所以要持续接触。

大脑原则7：睡眠（SLEEP | Rule #7: Sleep well, think well.）

睡眠更充足，大脑的运作就愈顺畅，因为它会有更多时间消化你所学习到的东西。如果在工作中能够小睡片刻，你的思路就会更清晰。

大脑原则8：压力（STRESS | Rule #8: Stressed brains don’t learn the same way.）

长期压力会造成不良影响，因为它会减弱你清晰思考的能力。要尽可能减轻自己的压力，生活中若有某个层面的压力，一定会扩散到所有层面中。

大脑原则9：感官（SENSORY INTEGRATION | Rule #9: Stimulate more of the senses.）

感官一向是共同运作的，要有更好的记忆力，就要在你想学习的内容中加入更多感官线索，影像、声音、气味、触感和味道，都是有效思考的催化剂。

大脑原则10：视觉（VISION | Rule #10: Vision trumps all other senses.）

毫无疑问地，视觉绝对是最有力的感官，在自己要学习的内容中加入图像，就可以多记住55％以上的信息，图像的学习效果总是胜过文字阅读。

大脑原则11：性别（GENDER | Rule #11: Male and female brains are different.）

男性与女性的大脑并不相同，它们以不同的方式处理情感。要帮助人学习到更多东西，要注意到所要传达的内容中，哪些部分能激发情感。

大脑原则12：探索（EXPLORATION | Rule #12: We are powerful and natural explorers.）

人类天生就会探索，我们喜欢自己提出假设、实验、观察然后得出结论。要运用这点特性，可以在演示文稿中加入探索的成分。

虽然很多原则以前就知道，但有一些新的观点可以学习，比如作者在书中提到，大脑是在持续动态的环境下发展进化来的，那么也该在动态环境下才能得到最好发挥。结论是cubicles有违天性，开会最好是散步中进行，而小学中学应该每天六个小时体育课－－知识理论课程不过是点缀。听起来很偏激但如果学校按这个理论来改进，上学会成为一件非常美好的事。
特别有意思的，书中提到在不同的运动里，aerobics是对大脑认知能力最有助益，toning就一般般了。还有就是，运动对记忆力帮助不大．本人对这一点持怀疑态度，运行跟记忆在目前的情况很难得出直接的证明．要证明这一点，需要大量的实验、研究和统计调查．不同的人在不同的时间、环境下，从事不同的运行与运动的时间跟记忆不同类型的东西这之间的关系是非常的复杂，想搞清楚可不是一件容易的事．

我自己的观点认为，运动与学习的关系是非常重要的，注意力以及学习的时间的长短与身体素质有直接的关系，而提高身体素质最好的方法就是运动．
