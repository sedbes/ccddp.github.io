一个简单的学习类型测试

时间：2023-02-08 20:20:31  来源：战隼  作者：

　　前段时间发了一个跟学习类型的文章：你知道自己的学习类型么？ ，今天找到一个小测试，非常的简单，只要几分钟就ＯＫ了。跟上面的文章不同的是，它把学习类型分为四种：视觉型 / 经验型 / 阅读型 / 听觉型。开始测试吧

　　请对下面5个问题作答，用数字1-4对每组的4个答案进行打分，4分表示该答案与您自己的情况最相符，1分则表示该答案与您的情况最不相符。

　　1、 当您计划给您的家人一个惊喜，为他们烹饪一盘新口味的菜式时， 您会：

　　A　___ 翻看菜谱， 试图从菜谱上的图片中寻找灵感

　　B　___ 不依靠任何意见，自己试着做

　　C　___ 参考菜谱目录内容

　　D、___ 向朋友讨教一个好的配方

　　2、 在安装一部新的机械设备时，您会选择：

　　A　_ 依据用户手册上的图示进行安装/B。_ 直接试着安装 – 边干边摸索

　　C　___ 安装前先阅读用户手册的文字说明

　　D　___ 向别人请教安装方法

　　3、 当有一位朋友请您给他们介绍您汽车上的发动机的工作原理时， 您会：

　　A　___ 画个图表以辅助说明您的描述

　　B　___ 讲述时，经常用手势强调您的话

　　C　___ 把您所讲述内容的关键字和关键句子写下来

　　D　___ 不借助任何辅助手段，只是口头讲述

　　4、 请试着回忆一下您最近一次学习如何去做一件新的事情的情形。 您的最佳学习方法是什么？

　　A　___ 参看图解、表格和图片

　　B　___ 直接试着动手操作

　　C　___ 阅读文字说明或使用说明书

　　D　___听别人讲解

　　5、 您的最佳记忆方法是：

　　A　___ 在大脑中描绘一幅图画。

　　B　___ 进行尝试或实践。

　　C　___ 阅读相关知识。

　　D　___ 听取相关介绍。 　　请计算您的全部答案得分，然后继续。
