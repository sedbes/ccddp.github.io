如何高效地学习知识？学会划分知识管理四象限

时间：2023-02-08 20:20:31  来源：其它  作者：个人发展

信息爆炸的时代，没有自己的知识管理体系，就无法甄别知识，无法消化知识，无法运用知识。

 

如何管理自己的知识体系

第一步，梳理自己的知识体系

个人知识状况可概括为四种：

意识

知道

个人知识状况可概括为四种：不知道自己知道，知道自己知道，不知道自己不知道，知道自己不知道

  1. 知道自己知道的知识，就是自己记得的并且经常使用的知识。特点是随叫随到，马上记起来。

  2. 不知道自己知道的知识，就是自己曾经学习和掌握的，但是时间久了忘记的知识，他们需要花点时间复习和回忆。

  3. 不知道自己不知道的知识，就是自己真的完全没听过的知识。

  4. 知道自己不知道的知识，就是自己知道有这方面的知识，但是没学进去、没掌握。

知道自己知道——多用，不知道自己知道——复习，不知道自己不知道——阅读和发现，自己不知道——坚持学习。

下面的解释有些抽象和啰嗦，故作以上理解和概括。

  1. 第一类知识，是自己知道的知识。比如语文数学、穿衣搭配、柴米油盐等通用型知识。这类知识重点不在学习，而是对其进行合理保存，以便在需要的时候能够找到它们。

  2. 第二类知识，是不知道自己曾知道的知识。这类知识属于已掌握但很少用到，因而没意识到，比如哺育小孩就仿佛是女性天生具备的能力。要对这类知识进行盘点、整理，将其转化为知道自己知道的知识。

  3. 第三类是我们不知道自己不知道的知识。对于这类完全未知的知识，需要广泛学习。

  4. 第四类知识是我们知道自己不知道的知识。比如80%以上的股民都是韭菜被收割，许多人清楚自己老亏钱是因为股票分析技术不过硬，但因学习难度大而放弃了，心存侥幸瞎买卖。对于知道自己不知道的知识，不要怕难而偷懒而不去学习。

一句话总结，知道自己知道——保存；不知道自己知道——盘点；不知道自己不知道——发现；知道自己不知道——重点。

第二步，划分时间价值四象限

时间价值四象限帮助我们评估应该学习什么知识：

高

时间少价值高

时间多价值高

低

低

时间少价值低

时间多价值低

少

时间

多

  1. 时间多价值高的知识，就是一门学科，需要长期坚持、循序渐进地学习。

  2. 时间少价值高的知识，就是一个章节，需要花点时间、集中尽力地学习。

  3. 时间少价值低的知识，就是可以快速浏览的、内容深度极低的资讯或小知识等。

  4. 时间多价值低的知识，就是自己肯定无法驾驭或精通，但是又非常想精通的知识。

  1. 第一种情况是需要花费时间多、获得价值高的知识。这种情况是时间成本和收获价值基本对等。面对这种知识，我们要斟酌是否存在更优的学习方案来。比如考研，但要花一年备考。许多考生会选报培训班。因为考研机构有考试经验、题库、优秀老师，还可以节省时间。

  2. 第二种情况是需要花费时间少、获得价值高的知识学习。这种情况是时间成本低，收获价值高。比如考试临时抱佛脚掌握老师所划出的重点。

  3. 第三种情况是需要花费时间少、获得价值低的知识学习。这种情况是时间成本低，收获价值也低，可以在休闲的时候用来消遣打发时光。比如朋友圈的很多文章。但这种消遣式的学习会不知不觉地占用大量碎片化时间，需要警惕。

  4. 第四种情况是需要花费时间多、获得价值低的知识学习。这种情况是时间成本高，收获价值低，建议规避。比如普通人沉迷游戏。

什么样的知识是有价值的知识？如果知识能够帮助你在升职加薪的同时也能带来愉悦和满足感，那这就是有价值的知识。
